Nomade Finance is a crypto exchange platform that allows users to buy and sell crypto assets with ease using their local currency. The client is built with React, Next.js, TailwindCSS, and TypeScript.

[Preview](https://nomade.finance), [Source code](https://github.com/Nomade-Finance/nomade-client).

### Stack
- [React](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
- [React Slick](https://expo.dev) - a popular library for building carousels (slideshows or image galleries) in a React application.
- [TypeScript](https://www.typescriptlang.org) - TypeScript is a language for application-scale JavaScript.
